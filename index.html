Spatial Mask Merging (SMM)

Paper-faithful ILP-based correlation clustering for spatial instance mask fusion

Overview

Spatial Mask Merging (SMM) is an open-source implementation of ILP-based correlation clustering for merging overlapping instance segmentation masks across tiled detections. Designed for remote sensing and aerial imagery research, SMM provides exact optimization and evaluation tools.

Repository

Visit the full source code and documentation on GitHub:
github.com/mihajlov39547/spatial-mask-merging

Current Release

v0.1.0-alpha — Early Experimental Release
Prototype version for transparency and early feedback.

Features

  • Exact ILP correlation clustering for instance mask merging
  • Evaluation tools: F1, PQ, Dice, and fragmentation metrics
  • Visualization and hyperparameter optimization utilities
  • Open MIT License for research and commercial use
